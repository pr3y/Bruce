Alterar ID (VID/PID):
VendorID e ProductID são códigos usados ​​para identificar seu teclado. SOMENTE se você estiver mirando em um macOS, lembre-se de inserir o comando ID , para alterar VID / PID, no topo do seu código.
* Você pode definir um ID personalizado do dispositivo USB HID do Flipper. O comando ID deve estar na primeira linha do script, ele é executado antes de rodar o script. A menos que seu alvo seja um macOS (que exige especificar na primeira linha do payload), VID e PID são códigos hexadecimais e são obrigatórios, Fabricante e Produto são strings de texto e são opcionais.
Remark (REM):
REM é a abreviação de Remark e adiciona um comentário ao payload, como um título, o sistema operacional alvo, o funcionamento de um comando ou o nome do autor.
String (STRING/ALTCODE):
STRING injeta pressionamentos de teclas, ou "digita", os caracteres fornecidos (a-z, 0-9, pontuação e caracteres especiais).
ALTCODE é usado em vez de STRING, para imprimir a string de texto usando o método "ALT+Numpad". Funciona mesmo que um layout de teclado não americano (non-US) esteja selecionado.
Delays:
DELAY é expresso em milissegundos (1 seg. = 1000 ms.). O mínimo é 100 (0,1 seg.). Aumenta/diminui em 100 milissegundos a cada clique.
DEFAULT_DELAY / DEFAULTDELAY adiciona um atraso antes de cada novo comando.
Comandos diversos:
REPEAT é usado para repetir o comando anterior. Ele espera o número de repetições adicionais como parâmetro.
ALTCHAR é para exibir teclas ALT+Numpad individuais (no Windows), ou teclas ALT(ALT+SHIFT) (no macOS).

Teclas modificadoras:
GUI é usado para ambos: a tecla WINDOWS(⊞) no Windows e a tecla CMD(⌘) no macOS. (No Windows, você pode usar a tecla WINDOWS ou a tecla GUI para obter o mesmo resultado). O mesmo vale para CONTROL e CTRL Se você precisar usar mais de uma "tecla modificadora" ao mesmo tempo, use as "Combinações de Teclas" pré-carregadas, que você encontrará logo abaixo dessas teclas modificadoras.
Combinações de Teclas:
APENAS no Windows: CTRL-ALT é o mesmo que digitar ALTgr (ALT Gráfico), e você pode usar tanto ALT-WINDOWS quanto ALT-GUI, e WINDOWS-SHIFT ou GUI-SHIFT para obter exatamente o mesmo resultado.
